The Interview Structure

The 16-Turn Framework.

Every interview is capped at 16 conversational turns, ensuring a highly focused, perfectly timed assessment that respects the candidate's time while extracting maximum technical signal.

Turns 1-3

Context & Calibration

The avatar welcomes the candidate, verifies their audio setup, and asks an ice-breaker based on their most recent CV role to establish a comfortable baseline.

Turns 4-10

Technical Deep Dive

The LLM dynamically generates complex, multi-part technical architecture questions. It listens to the candidate's response and actively probes weak points.

Turns 11-14

Behavioral STAR

Shifting to soft skills, the AI asks situational questions (Situation, Task, Action, Result) to evaluate leadership, conflict resolution, and teamwork.

Turns 15-16

Conclusion & Scoring

The candidate asks any final questions about the role. The session gracefully closes, and the `/api/evaluate` endpoint immediately fires the scorecard.

Built For Scale

Enterprise-Grade Architecture.

Tenkir AI isn't just a wrapper—it's a massive orchestration of real-time technologies. We process audio chunks, map facial blendshapes, and stream LLM tokens simultaneously in the browser to achieve sub-200ms perceived latency.

React Three Fiber & WebGL

Hardware-accelerated 3D rendering creates a pristine corporate office environment directly in the browser, with dynamic lighting and shadow casting.

Web Audio API & ARKit

An Audio AnalyserNode maps raw streaming voice frequencies to 52 precise ARKit facial blendshapes for flawless, procedural lip-syncing without pre-rendering.

Next.js & Supabase

A serverless backend architecture utilizing Next.js API routes ensures infinite scalability, backed by Supabase for strict Row-Level Security (RLS) on all candidate data.

app/api/evaluate/route.ts 
// 1. Analyze 16-turn transcript
const scorecard = await llm.evaluate({
  resume: candidate.cvText,
  transcript: session.messages,
  rubric: role.technicalRubric
});

// 2. Save securely to Supabase
await supabase.from('evaluations').insert({
  candidate_id: candidate.id,
  score: scorecard.overallScore,
  technical_depth: scorecard.techDepth
});

// 3. Dispatch HR Email via Resend
await resend.emails.send({
  to: recruiter.email,
  subject: `Tenkir Evaluation: ${candidate.name}`,
  react: ScorecardTemplate({ scorecard })
});
